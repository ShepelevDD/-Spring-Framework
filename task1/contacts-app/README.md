Данная программа является небольшим консольным приложением "Контакты".

Программа запускается либо через IDE, поддерживающее работу с языком Java (Например, IntelliJ IDEA), либо через консоль.
Старт программы начинается при запуске метода main класса Main.

После запуска метода main в консоль выведутся опции:

0 - Завершить работу приложения. При выборе этой опции программа завершит свою работу.

1 - Вывести имеющиеся контакты. При выборе этой опции программа выведет на консоль список имеющихся контактов или
уведомит об отсутствии контактов в списке

2 - Добавить новый контакт. Выбор данной опции позволит Вам добавить в список контактов новый контакт. Для этого
программа через сообщения в консоли предложит ввести сначала Ф.И.О., затем номер телефона и email контакта. При введении
пустой строки программа через сообщение в консоли попросит снова ввести данные для контакта, который Вы хотите добавить

3 - Удалить контакт по email. Выбор данной опции позволит Вам удалить контакт из списка контактов, указав email контакта,
который Вы хотите удалить.

4 - Сохранить имеющиеся контакты. Данная опция позволит Вам сохранить список контактов в текстовый файл "contacts.txt", 
место хранения которого можно изменить в файле "application.properties", поменяв значение параметра "app.save.path" (по
умолчанию в данном параметре указан путь "src/main/resources/contacts.txt").

Также имеется возможность при старте работы приложения подгрузить уже имеющийся список контактов. Для этого в файле
"application.properties" присвойте параметру "spring.profiles.active" значение "init" (по умолчанию это значение
равняется "default"). Все контакты должны быть разделены переносом строки. Пример файла:

Иванов Иван Иванович;+890999999;someEmail@example.example \
Смирнов Иван Иванович;+890999991;someEmail2@example.example

Сам путь к файлу с хранилищем контактов нужно указать в файле "application-init.properties" в параметре "app.init.path"
(по умолчанию в данном параметре указан путь "src/main/resources/default-contacts.txt")
